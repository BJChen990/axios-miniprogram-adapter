## megalo-demo

megalo demo 工程，支持微信小程序和支付宝小程序，在逐步完善中。

## 运行

微信小程序版本：

```bash
npm run dev:wechat
```

支付宝小程序版本：

```bash
npm run dev:alipay
```

百度智能小程序版本：

```bash
npm run dev:swan
```
## 注意，demo里请求用到了下列域名，如需在真机上运行，请将其加入安全域名设置
- suggest.taobao.com
- api.github.com
- easy-mock.com
